To test the system, you can send an alert message to the Orchestrator by using:

kubectl port-forward svc/{{ include "observability-agent.fullname" . }}-nats 4222:4222
# Use nats CLI or another NATS client to publish messages

To access the UI:

kubectl port-forward svc/{{ include "observability-agent.fullname" . }}-ui 8080:80
# Then open http://localhost:8080 in your browser

To access the UI Backend API directly:

kubectl port-forward svc/{{ include "observability-agent.fullname" . }}-ui-backend 5000:5000
# Then use http://localhost:5000/api endpoints

To access the K8s Command Backend API directly:

kubectl port-forward svc/{{ include "observability-agent.fullname" . }}-k8s-command-backend 5002:5002
# Then use http://localhost:5002/api endpoints
